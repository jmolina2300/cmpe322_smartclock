Description:

  A Smart clock that uses an ESP8266 microcontroller.

  Our final project for CMPE322 Microcontrollers and Interfacing.
  
Contributors:

  - London Johnson
  - Jan Molina
  - Kyle Smith


Hardware Requirements:
 
  ESP8266, LOLIN WEMOS D1 mini pro (V1)
  Adafruit 7-Segment Display with I2C Backpack
  DS3231 RTC Module
 
Pin connections:
 
  D1 - SCL
  D2 - SDA
  D3 - Status LED 
  D4 - Buzzer
  D5 - button - Decrement
  D6 - button - Increment
  D7 - button - Mode/snooze
